Vltur配置ipv6下的shadowscok

1、Why vultr vps?

  • 穷！

Digtalocean的优惠用完了，恰好看见极客电台更新了vultrde的优惠信息，比DO力度更大，同时也支持ipv6

2、注册的过程

1.首先是打开vultr官网，免费注册一个vultr账号。填写邮箱地址、密码，非常简单。如果你图安全，你还能开启两步验证功能，每次登录需要启用google authenticator验证码。

2.注册登录后台，可看左侧面板有一列功能栏，从上至下顺序分别是Servers服务器列表、Billing财务中心、Support客服中心、Affiliate推广、Account账户中心。

3.最常用的是billing，你要先充值，推荐使用信用卡充值，没有信用卡的用户使用Paypal充值，可自主选择充值金额。

4.Vultr目前提供机房位置有：美洲、欧洲、澳大利亚、亚洲日本东京机房。

5.接下来是服务器软件参数，1G以上内存推荐安装64位系统，最大程度发挥服务器性能。主流Linux系统推荐Ubuntu，社区教程多，有问题可自行google解决。CentOS\Debian也受技术党的喜爱。如果你是Windows粉丝，Vultr当然也提供windows server 2012 R2版本，价格略高一些，商业系统嘛！

我们选择CentOS 7

一定要勾选的：

  • Enable IPv6：启用服务器IPv6支持;

  • SSH Keys：使用ssh keys登录管理服务器

配置ssh登录：https://www.zxc.so/do-ssh-key.html
（强烈推荐安装putty和winscp）

这些看着办

  • Enable Private Network：用于企业大规模部署内部服务器私有资源；

  • Enable Auto Backups：启用服务器自动备份，对个人用户来说不需要，每月额外收费1美元；

  • Enable DDOS Protection：启用DDOS保护功能，个人用户同样不需要；

  • startup Script：开机自动脚本；适合大规模部署服务器，使用同样的安装脚本，个人用户不需要。

  • Server Label：给你的服务器取名字，随便填啥都无所谓，给自己看的。

点击最底部的Deploy，就会自动安装服务器啦！一般五分钟后，你的邮箱就会收到Vultr开通邮件，提醒你服务器安装完成。

点击后台你搭建的服务器，出现该vps的控制面板：这里你可以找到服务器的ip地址等等

找到你的ipv6地址和ipv4地址，并备份下来！

Troubeshooting

1、配置完成，shadowsock配合chrome的插件代理，并不能有效上网

google一圈之后，很可能是系统防火墙的原因，限制了端口

系统 CentOS 7 x64
懒得去查太多的设置，简单粗暴，直接关闭防火墙

centos 7版本以后默认使用firewalld后，网上关于iptables的设置方法已经不管用
官方文档地址：https://access.redhat.com/documentation/en-US/Red_Hat_Enterprise_Linux/7/html/Security_Guide/sec-Using_Firewalls.html#sec-Introduction_to_firewalld

这里使用到两步：
1-安装firewalld root下执行：

# yum install firewalld firewall-config

2-关闭firewall

# systemctl disable firewalld
 禁用：# systemctl stop firewalld

可以不用重启
